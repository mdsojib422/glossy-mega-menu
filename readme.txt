Glossy Mega Menu is a WordPress plugin for creating stunning mega menus with Elementor, including a header and footer builder for design flexibility.

== Description ==
Glossy Mega Menu
Transform your existing WordPress menu into a fully customizable mega menu with Glossy Mega Menu! Designed to work seamlessly with Elementor, this plugin offers an easy, intuitive way to enhance your website's navigation. Add WordPress widgets, style your menu using the Elementor editor, and configure settings to suit your needs—all with a few clicks. Glossy Mega Menu is perfect for anyone looking to create a user-friendly, touch-responsive, and visually stunning navigation experience.

Features
    ●	Easy Integration with Elementor: Build and customize your mega menu using the Elementor page builder.
    ●	Fully Responsive: Automatically adjusts to all screen sizes, ensuring a consistent user experience on mobile, tablet, and desktop devices.
    ●	Drag & Drop Interface: Easily add and organize menu items, create rows and columns, and insert widgets using a simple drag-and-drop    interface.
    ●	Customizable Styling: Tailor your menu’s appearance to match your website design with Elementor's styling options.
    ●	Mobile-Friendly Navigation: Includes options for off-canvas mobile menus, collapsible submenus, and responsive breakpoints.
    ●	Multiple Menu Locations: Supports multiple menu locations, each with unique configurations.
    ●	Touch and Keyboard Accessible: Built with accessibility in mind, providing native touch support and keyboard navigation for a seamless experience.

== Installation ==
Go to your WordPress Dashboard.
Navigate to Plugins > Add New.
Search for “Glossy Mega Menu” and click Install Now.
Once installed, click Activate to enable the plugin.

Mega Menu Creation Guideline:
1. Appearance > Menus > Select Menu > Enable Mega Menu > Edit Mega Menu > save Mega menu
2. Appearance > Header & Footer Builder > Add New Header > Select Types Of Template > Save
3. Edit With Elementor > Must Use GlossyMM Nav Menu (Required) And Save

You’re ready to start creating your custom mega menu with Elementor!
